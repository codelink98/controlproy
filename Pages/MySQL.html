MySQL:

MySQL es un SGBD muy popular, potente, confiable, de open source y con un equipo de desarrolo totalmente propio, tiene muchas herramientas que le dan ventaja sobre otros SGBD, en 2019 fue el SGBD del año, cuenta con opciones que se pueden adaptar a lo que necesites. Leer mas...

Caracteristicas :

MySQL utiliza un modelo de trabajo relacional, archiva datos en tabla en lugar de todo en un mismo archivo, lo que lo hace mas flexible y eficiente, las tablas se unen de formas definidas, por lo cual se pueden conectar. Es una aplicacion que cualquiera puede utilizar, facil de manejar,con dos versiones, comercial y open source, asi como todas las mencionadas. MySQL tiene mas caracteristicas como lo son:

  • Diferentes motores de almacenamiento
  • Agrupacion de transacciones
  • Seguridad
  • EJecucion de transacciones y uso de clave foraneas
  • Replicacion
  • Portable
  • Servicio de usuarios y contraseña
  • Soporta gran cantidad de datos
  • Se desarrollo en C y C++
  • Es gratuito
