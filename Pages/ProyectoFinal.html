Reporte final de sistema individual:

Nombre del sistema: Arte Frausto

Este sistema permite llevar y administrar una galeria, cuenta con los apartados de inventario, vender, vendidos, empleados y exposiciones.

Tecnologias utilizadas para la creacion de este sistema:

El sistema esta basado en web principalmente, corre nativamente en el localhost port 5000 y se ejecuta desde el navegador.

  • 1- Python Flask: Para el BackEnd del sistema se utilizo el framework "FLASK" basado en python por su facilidad de uso y codigo facil de leer.
  • 2- Jinja 2: Jinja es un motor de plantillas HTML que facilita la muestra de informacion actualizada en el navegador
  • 3- HTML y Bootswatch: Para el FrontEnd y estructura de la pagina, sus vistas y estilo se utilizaron HTML y Bootswatch que es un framework basado en bootstrap que contiene estilos predefinidos.
  • 4- MariaDB: Como motor de base de datos se utlizo MariaDB igual que durante el curso de esta clase.

Vistas del sistema:

Index/Vista principal

Aqui se muestra el inventario, el menu para agregar nuevas obras, las opciones para manipular los datos del inventario asi como la opcion de buscar dentro del inventario

Vender

Aqui se muestran el inventario y el apartado buscar al igual que en la parte anterior pero ahora cada entrada cuenta con un boton "vender" que envia a la pantalla de facturacion para realizar la venta de la obra.

Vendidos

Aqui se muestra el historial de obras vendidas y un menu de busqueda en el historial asi como debajo de el historial se muestran el historial de facturas con los datos y un menu de buscar entre ellas.

Empleados

En la ventana de empleados se muestran los empleados registrados asi como un menu para modificarlos, eliminarlos o agregar nuevos.

Exposiciones

Aqui se muestran las obras que estan actualmente en exposicion asi como las que se pueden agregar.
