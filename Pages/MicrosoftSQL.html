Microsoft SQL server:

Caracteristicas:

Microsoft SQL es un SGBD que tiene soporte para transacciones es estable y seguro, realmente una buena opcion para la gestions de Datos permite varios modos de trabajo como el "cliente servidor", entorno grafico amigable, DDL, DML grafico, permite gestionar varios servidores de Datos y cuenta con una version gratuita llamada SQL style Express Edition.

  • Inteligencia en todos tus Datos
  • Lenguaje a eleccion del usuario
  • Rendimiento Lider
  • Plataforma segura
  • Rapida y sustentable
  • Bi con todos tus archivos

SQL Server 2019: características | Microsoft. (s. f.). Microsoft. 12 de septiembre de 2020, de https://www.microsoft.com/es-es/sql-server/sql-server-2019-features

Tipos de Datos:

Los tipos de datos en Microsoft SQL son los siguientes:

  1. Numericos Exactos (int,Smallint,bit,money,etc)
  2. Numericos Aproximados (Float,real)
  3. Fecha y hora (DateTime,SmallDateTime)
  4. Cadenas de Caracteres (Char,Varchar,text)
  5. Cadenas de Caracteres Unicode(nChar,nVarchar,ntext)
  6. Cadenas Binarias (Binary,varBinary,image)

Tipos de datos en SQL Server - Base de Datos. (s. f.). Google Sites. 12 de septiembre de 2020, de https://sites.google.com/site/basdededatosrelacionales/home/contenido/tipos-de-datos-en-sql-server

Requerimientos:

De los 3 SGDD que hemos tratado el dia de hoy Microsoft SQL Server tiene los requerimientos mas amigables:

  • Windows 7 minimo
  • Hard Disk: minimum of 6GB de espacio en disco duro.
  • Monitor: 800x600.
  • Acceso a Internet
  • RAM minima de 1GB
  • Procesador minimo: 1.4GHz AMD o Intel
